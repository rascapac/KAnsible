# KAnsible


### Description
Ansible playbook to deploy important tools that Kali Linux is missing. 

### How to install locally
```sh
git clone https://github.com/M507/Kali-TX.git
cd KAnsible
sudo bash install.sh
```

### Deploys

- Docker
- Empire
- Dirsearch
